# 八字可视化模块整合总结

## 整合目标

统一四柱计算模块，使八字可视化模块使用八字排盘模块作为四柱计算的唯一入口。

## 现状分析

### 之前的模块结构
- **四柱计算器.py** - 混合使用 lunar_python 和 sxtwl，有备用算法
- **高精度四柱计算器.py** - 仅使用 lunar_python（建议删除）
- **高精度农历转换.py** - 双库校验，推荐作为唯一入口
- **八字排盘模块.py** - 已依赖高精度农历转换

### 整合后的结构
- **八字排盘模块.py** - 统一的四柱计算入口
- **八字可视化.py** - 统一使用八字排盘模块
- **其他模块** - 通过八字排盘模块获取四柱数据

#### 2.1 show_bazi_from_birth 函数

**功能**: 从出生信息直接显示八字排盘（推荐方式）

**参数**:
- `parent`: 父窗口
- `year`: 出生年
- `month`: 出生月
- `day`: 出生日
- `hour`: 出生时（默认12）
- `minute`: 出生分（默认0）
- `gender`: 性别（默认'男'）
- `analysis_data`: 分析数据（可选）

**返回值**: BaZiDialog 对话框对象

**示例**:
```python
dialog = show_bazi_from_birth(root, 1990, 5, 15, 10, 30, '男')
# 获取排盘数据
panpan_data = dialog.visualizer.panpan_data
```

## 使用方式

### 方式一：使用便捷函数（推荐）

```python
from modules.八字可视化 import show_bazi_from_birth

# 显示八字排盘对话框
dialog = show_bazi_from_birth(
    parent=root,
    year=1990,
    month=5,
    day=15,
    hour=10,
    minute=30,
    gender='男'
)
```

### 方式二：手动创建并显示

```python
from modules.八字排盘 import create_bazi_panpan
from modules.八字可视化 import BaZiDialog

# 创建八字排盘对象
bazi = create_bazi_panpan(1990, 5, 15, 10, 30, '男')
panpan_data = bazi.get_panpan_result()
dayun_list = bazi.get_dayun()

# 显示对话框
dialog = BaZiDialog(root, panpan_data, analysis_data, dayun_list)
```

### 方式三：快速显示

```python
from modules.八字可视化 import quick_show_bazi

# 快速显示（不使用分钟）
dialog = quick_show_bazi(root, 1990, 5, 15, 10, '男')
```
