# 专业级正五行择日软件 - 打包说明

## 快速开始

### 方法一：使用打包脚本（推荐）

```bash
# 一键打包
python build_exe.py

# 清理并重新打包
python build_exe.py --clean

# 跳过测试直接打包
python build_exe.py --skip-tests
```

### 方法二：使用 PyInstaller 直接打包

```bash
# 安装 PyInstaller
pip install pyinstaller

# 使用配置文件打包
pyinstaller 专业级正五行择日软件.spec --clean

# 打包完成后，可执行文件位于 dist/专业级正五行择日软件.exe
```

## 打包配置说明

### .spec 文件配置

`专业级正五行择日软件.spec` 文件包含以下主要配置：

#### hiddenimports（隐藏导入）

已配置所有必要的模块：

- **核心模块**: 高精度农历转换、四柱计算器、评分器等
- **八字模块**: 八字排盘、八字可视化、八字分析工具等
- **神煞模块**: 嫁娶、安葬、修造、作灶等神煞检查器
- **规则模块**: 各事项的宜忌规则检查器
- **第三方库**: lunar_python、loguru、tkinter 等

#### excludes（排除模块）

为减小体积，已排除以下模块：

- matplotlib、numpy、pandas、scipy（科学计算库）
- PIL（图像处理）
- PyQt5/6、wxPython（其他GUI库）
- test、unittest、pytest（测试框架）

#### 其他配置

- `console=False`: 不显示控制台窗口（GUI模式）
- `upx=True`: 使用 UPX 压缩减小体积
- `icon`: 可添加图标文件（需取消注释）

## 打包选项

### 单文件版本（默认）

- **优点**: 只有一个 .exe 文件，便于分发
- **缺点**: 启动稍慢，文件较大
- **配置**: 使用默认的 .spec 文件

### 单目录版本

- **优点**: 启动快，文件较小
- **缺点**: 需要分发整个目录
- **配置**: 修改 .spec 文件，使用 COLLECT 替代 EXE

```python
# 在 .spec 文件中注释掉 EXE 配置，取消 COLLECT 注释
coll = COLLECT(
    exe,
    a.binaries,
    a.zipfiles,
    a.datas,
    strip=False,
    upx=True,
    upx_exclude=[],
    name='专业级正五行择日软件'
)
```

## 常见问题

### 1. 打包后程序无法启动

**可能原因**: 缺少依赖模块

**解决方法**:
- 检查 .spec 文件的 hiddenimports 是否包含所有必要模块
- 使用 `--debug=all` 参数重新打包查看错误信息

### 2. 文件体积过大

**解决方法**:
- 确保 excludes 列表包含不需要的模块
- 使用 UPX 压缩（已默认启用）
- 考虑使用单目录版本

### 3. 杀毒软件误报

**解决方法**:
- 这是 PyInstaller 打包程序的常见问题
- 将程序添加到杀毒软件白名单
- 考虑使用代码签名证书

### 4. 图标不显示

**解决方法**:
- 准备 .ico 格式的图标文件
- 在 .spec 文件中取消 `icon='icon.ico'` 的注释
- 确保图标文件路径正确

### 5. 中文显示乱码

**解决方法**:
- 确保 .spec 文件第一行包含 `# -*- coding: utf-8 -*-`
- 在程序中正确设置中文字体
