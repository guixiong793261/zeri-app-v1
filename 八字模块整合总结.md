## 五、核心功能演示

### 5.1 事主八字分析

```python
from modules.事主八字分析 import ShiZhuBaZiAnalyzer

analyzer = ShiZhuBaZiAnalyzer(1984, 2, 15, 10, 0, '男')
result = analyzer.get_analysis_result()

print(f"日主：{result['日主信息']['日主']}")
print(f"旺衰：{result['旺衰分析']['旺衰等级']}")
print(f"用神：{result['喜用神']['用神']}")
```

**输出**:
```
日主：己(土)
旺衰：偏弱
用神：['火']
```

### 5.2 日课匹配评分

```python
rike_sizhu = {
    'year_gan': '甲', 'year_zhi': '辰',
    'month_gan': '丙', 'month_zhi': '寅',
    'day_gan': '戊', 'day_zhi': '辰',
    'hour_gan': '丙', 'hour_zhi': '午'
}

match_result = analyzer.calculate_rike_match_score(rike_sizhu)
print(f"匹配分数：{match_result['score']}分")
print(f"匹配等级：{match_result['level']}")
```

**输出**:
```
匹配分数：67.0分
匹配等级：吉
```

### 5.3 多个事主综合评分

```python
from modules.事主日课匹配评分 import calculate_multi_match

shizhu_list = [
    {'name': '新郎', 'year': 1984, 'month': 2, 'day': 15, 'hour': 10, 'gender': '男'},
    {'name': '新娘', 'year': 1986, 'month': 5, 'day': 20, 'hour': 14, 'gender': '女'}
]

result = calculate_multi_match(shizhu_list, rike_sizhu)
print(f"综合评分：{result['score']}分")
print(f"综合等级：{result['level']}")
```

**输出**:
```
综合评分：46.3分
综合等级：中平
```
