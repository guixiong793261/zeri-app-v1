# 八字工具模块整合说明

## 一、整合目标

统一所有八字相关工具函数，避免重复代码，提供一致的数据定义和接口。

## 三、迁移指南

### 3.1 新开发模块

**推荐做法**：
```python
# 统一从整合模块导入
from modules.八字工具整合 import (
    GAN_WUXING, ZHI_WUXING,  # 五行
    GAN_YINYANG, ZHI_YINYANG,  # 阴阳
    ZHIGAN_WEIGHTED,  # 藏干
    get_shishen,  # 十神
    get_zhangsheng,  # 十二长生
    check_liuhe, check_liuchong,  # 地支关系
    get_nayin,  # 纳音
    calculate_wuxing_score,  # 五行分数
)
```

### 3.2 旧模块迁移

#### 工具函数.py 的迁移

**当前导入方式**：
```python
from modules.工具函数 import TIAN_GAN_WUXING, get_shishen
```

**建议迁移为**：
```python
from modules.八字工具整合 import GAN_WUXING, get_shishen

# 保持兼容性
TIAN_GAN_WUXING = GAN_WUXING
```

**需要替换的函数**：
- `get_shishen()` → 使用新版基于阴阳的算法（接口不变）

#### 喜用神计算器.py 的迁移

**当前导入方式**：
```python
from modules.喜用神计算器 import ZHIGAN_MAP, calculate_xishen_yongshen
```

**建议迁移为**：
```python
from modules.八字工具整合 import ZHIGAN_WEIGHTED, ZHIGAN_SIMPLE
from modules.喜用神计算器 import calculate_xishen_yongshen

# 在喜用神计算器中修改
ZHIGAN_MAP = ZHIGAN_SIMPLE  # 或 ZHIGAN_WEIGHTED
```

#### 八字分析工具.py 的迁移

**当前导入方式**：
```python
from modules.八字分析工具 import (
    GAN_WUXING, ZHI_WUXING,
    ZHIGAN_MAP,
    get_shishen,
    get_zhangsheng,
    # ...
)
```

**建议迁移为**：
```python
from modules.八字工具整合 import (
    GAN_WUXING, ZHI_WUXING,
    ZHIGAN_WEIGHTED,
    get_shishen,
    get_zhangsheng,
    # ...
)
```

## 四、数据定义对照

### 4.1 藏干定义对照

**简化版**（旧版）：
```python
ZHIGAN_MAP = {
    '子': ['癸'],
    '丑': ['己', '癸', '辛'],
    '寅': ['甲', '丙', '戊'],
    # ...
}
```

**加权版**（新版）：
```python
ZHIGAN_WEIGHTED = {
    '子': [('癸', 1.0)],
    '丑': [('己', 0.6), ('癸', 0.3), ('辛', 0.1)],
    '寅': [('甲', 0.6), ('丙', 0.3), ('戊', 0.1)],
    # ...
}
```

**建议**：
- 需要精确计算时使用加权版
- 仅需列表时使用简化版（通过 `ZHIGAN_SIMPLE` 获取）

### 4.2 十神计算对照

## 五、API 参考

### 5.1 基础查询

```python
# 五行查询
get_gan_wuxing('甲')  # 返回: '木'
get_zhi_wuxing('寅')  # 返回: '木'

# 阴阳查询
get_gan_yinyang('甲')  # 返回: '阳'
get_zhi_yinyang('寅')  # 返回: '阳'

# 藏干查询
get_canggan('寅', weighted=True)  # 返回: [('甲', 0.6), ('丙', 0.3), ('戊', 0.1)]
get_canggan('寅', weighted=False)  # 返回: ['甲', '丙', '戊']
```

### 5.2 十神计算

```python
# 计算十神
get_shishen('甲', '乙')  # 返回: '劫财'（同我异性）
get_shishen('甲', '丙')  # 返回: '食神'（我生同性）

# 获取解释
get_shishen_explanation('食神')  # 返回十神解释
```

### 5.3 十二长生

```python
# 获取长生状态
get_zhangsheng('甲', '亥')  # 返回: '长生'
get_zhangsheng('甲', '卯')  # 返回: '帝旺'

# 获取力量系数
get_zhangsheng_power('帝旺')  # 返回: 1.0
get_zhangsheng_power('死')    # 返回: 0.3
```

### 5.4 地支关系

```python
# 六合
check_liuhe('子', '丑')  # 返回: '土'

# 六冲
check_liuchong('子', '午')  # 返回: True

# 六害
check_liuhai('子', '未')  # 返回: True

# 三刑
check_sanxing('子', '卯')  # 返回: '无礼之刑'

# 三合
check_sanhe(['申', '子', '辰'])  # 返回: '水'

# 三会
check_sanhui(['寅', '卯', '辰'])  # 返回: '木'
```

### 5.5 纳音

```python
get_nayin('甲子')  # 返回: '海中金'
get_nayin('丙寅')  # 返回: '炉中火'
```

### 5.6 五行计算

```python
# 计算五行分数
sizhu = {
    'year_gan': '甲', 'month_gan': '丙', 'day_gan': '戊', 'hour_gan': '庚',
    'year_zhi': '子', 'month_zhi': '寅', 'day_zhi': '辰', 'hour_zhi': '午'
}
scores = calculate_wuxing_score(sizhu)
# 返回: {'金': x, '木': x, '水': x, '火': x, '土': x}

# 五行关系
get_wuxing_relation('木', '火')  # 返回: '生'
get_wuxing_relation('木', '土')  # 返回: '克'
```

### 5.7 干支偏移

```python
# 干支偏移计算
gan, zhi = get_ganzhi_by_offset('甲', '子', 10)  # 返回: ('甲', '戌')
gan, zhi = get_ganzhi_by_offset('甲', '子', -10)  # 返回: ('甲', '寅')

# 偏移量计算
get_gan_offset('甲', '丙')  # 返回: 2
get_zhi_offset('子', '寅')  # 返回: 2
```
