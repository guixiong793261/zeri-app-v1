### 2.3 核心模块功能

#### 八字工具整合.py

**数据定义**：
- `TIAN_GAN`, `DI_ZHI` - 天干地支
- `GAN_WUXING`, `ZHI_WUXING` - 五行属性
- `GAN_YINYANG`, `ZHI_YINYANG` - 阴阳属性
- `ZHIGAN_WEIGHTED` - 加权藏干表
- `WUXING_SHENG`, `WUXING_KE` - 五行生克

**基础函数**：
- `get_gan_wuxing(gan)` - 获取天干五行
- `get_zhi_wuxing(zhi)` - 获取地支五行
- `get_shishen(day_gan, target_gan)` - 计算十神

**高级函数**：
- `calculate_wuxing_score(sizhu, ...)` - 计算五行分数（加权）
- `get_zhangsheng(day_gan, zhi)` - 计算十二长生
- `get_nayin(pillar)` - 计算纳音五行
- `check_liuhe(zhi1, zhi2)` - 检查六合
- `check_liuchong(zhi1, zhi2)` - 检查六冲
- `check_liuhai(zhi1, zhi2)` - 检查六害
- `check_sanxing(zhi1, zhi2)` - 检查三刑
- `check_sanhe(zhis)` - 检查三合
- `check_sanhui(zhis)` - 检查三会

#### 工具函数.py

**基础功能**（保留）：
- 基础数据定义（`TIAN_GAN`, `DI_ZHI`等）
- 基础函数（`get_gan_wuxing`, `get_shishen`等）
- 辅助函数（`format_date`, `get_hour_zhi_index`等）

**向后兼容**（新增）：

- 可以直接从`工具函数`导入`calculate_wuxing_score`等函数
- 保持现有代码无需修改即可使用新功能

#### 四柱计算器.py

**基础功能**（保留）：
- `calculate_sizhu()` - 计算四柱
- `get_lunar_date()` - 获取农历日期
- `analyze_sizhu()` - 基础分析

**增强功能**（新增）：

  - 十二长生状态
  - 纳音五行
  - 地支关系（六合、六冲、六害、三刑、三合、三会）
  - 五行分数（加权）
- `enhance_sizhu(sizhu)` - 增强sizhu字典

#### 喜用神计算器.py

**优化**（已更新）：

- 调整阈值（从整数改为浮点数）

## 四、使用示例

### 4.1 基础使用

```python
from modules.八字工具整合 import (
    get_gan_wuxing, get_zhi_wuxing,
    calculate_wuxing_score
)

# 获取五行
gan_wx = get_gan_wuxing('甲')  # 木
zhi_wx = get_zhi_wuxing('子')  # 水

# 计算五行分数
sizhu = {
    'year_gan': '甲', 'year_zhi': '子',
    'month_gan': '丙', 'month_zhi': '寅',
    'day_gan': '戊', 'day_zhi': '辰',
    'hour_gan': '庚', 'hour_zhi': '午'
}
scores = calculate_wuxing_score(sizhu)
print(scores)  # {'金': 1.0, '木': 2.25, '水': 1.35, '火': 2.3, '土': 2.3}
```

### 4.2 深度分析

```python
from modules.四柱计算器 import calculate_sizhu, enhance_sizhu
from datetime import date

# 计算并增强四柱
sizhu = calculate_sizhu(date(2024, 5, 20), 12, 0)
enhanced = enhance_sizhu(sizhu)

# 访问深度分析结果
analysis = enhanced['深度分析']
print("十二长生:", analysis['十二长生'])
print("地支关系:", analysis['地支关系'])
print("五行分数:", analysis['五行分数'])
```

### 4.3 喜用神计算
