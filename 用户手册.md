## 1. 软件概述

### 1.1 设计理念：五行为主，黄道为用

**专业级正五行择日软件**秉承传统择日学精髓，以"正五行择日法"为核心理论基础，结合现代计算机技术，为用户提供专业、准确、便捷的择日服务。

#### 核心理念

**"五行为主，黄道为用"**

- **五行为主**：以四柱八字的五行生克制化为核心判断标准
  - 补龙：日课五行补益山家（坐山）
  - 扶山：月令、日支扶助山家
  - 相主：日课与事主八字相合
  - 避凶：避开各种凶煞

- **黄道为用**：以大黄道、小黄道为辅助参考
  - 大黄道十二值星：青龙、明堂、金匮、宝光、玉堂、司命为吉
  - 小黄道十二建星：除、危、定、执、成、开为吉

#### 技术特色

1. **双库校验**：同时使用 sxtwl（天文算法）和 lunar-python（数据表）进行农历转换，确保四柱计算准确
2. **高精度计算**：精确到分钟的节气计算，正确处理立春、立夏等交界时刻
3. **系统化神煞**：涵盖年、月、日、时各类神煞，以及13类事项的专用神煞
4. **智能评分**：基于多维度因素的综合评分系统，结果直观易懂

### 1.2 主要功能

#### 13类事项择日

## 3. 快速入门

### 3.1 选择一个事项（嫁娶为例）

#### 步骤

1. 在左侧"事项选择"区域，点击 **"嫁娶"** 单选按钮
2. 界面会自动显示嫁娶事项的特殊选项

#### 嫁娶特殊选项

```
特殊选项：
  新娘生肖：[下拉选择，如：鼠、牛、虎...]
  新郎生肖：[下拉选择，如：鼠、牛、虎...]
  避开双方父母生肖冲日 [复选框]
```

> **提示**：不同事项会显示不同的特殊选项，如安葬需要选择山向，作灶需要选择灶向等。

### 3.2 设置日期范围

#### 步骤

1. 在"日期范围"区域，点击"从"后面的日期选择框
2. 在弹出的日历中选择开始日期（如：2026-03-01）
3. 点击"到"后面的日期选择框
4. 选择结束日期（如：2026-06-30）

#### 建议

- **时间跨度**：建议 3-6 个月，太短可能找不到合适日期，太长计算时间较长
- **提前期**：重要事项建议提前 3-6 个月开始择日
- **避开节气交界**：如需精确择日，注意避开立春、立夏等节气交界时刻

### 3.3 输入事主信息（可选）

#### 为什么要输入事主信息？

输入事主八字后，软件会：
- 分析事主喜用神
- 判断日课是否与事主八字相合
- 提供更个性化的择日建议

#### 步骤

1. 点击 **"添加事主"** 按钮
2. 在弹出的对话框中输入：
   - 姓名（可选）
   - 性别（男/女）
   - 出生日期（公历）
   - 出生时间（精确到小时）
3. 点击"确定"保存

#### 查看八字排盘

1. 添加事主后，在事主列表中选中该事主
2. 点击 **"八字排盘详情"** 按钮
3. 查看完整的八字分析，包括：
   - 四柱信息（年柱、月柱、日柱、时柱）
   - 十神分析
   - 五行分数
   - 纳音五行
   - 十二长生状态
   - 大运走势

> **注意**：事主信息为可选项。如果不输入，软件仍可进行择日，但不会进行八字匹配分析。

## 4. 功能详解

### 4.1 主界面布局

#### 界面分区

主界面分为四个主要区域：

1. **顶部标题栏**：软件名称、版本、帮助和关于按钮
2. **左侧选择区**：事项类型选择
3. **中间结果区**：择日结果列表
4. **底部设置区**：日期范围、特殊选项、事主信息、功能按钮

#### 界面操作技巧

- **调整窗口大小**：拖动窗口边缘可调整大小，软件会自动适应
- **列宽调整**：结果列表的列宽可以拖动调整
- **排序功能**：点击列表表头可按该列排序
- **快捷键**：
  - `F1`：打开帮助
  - `Ctrl+S`：导出结果
  - `Ctrl+R`：开始择日
  - `Esc`：关闭弹窗

### 4.2 事项类型与特殊选项

#### 各事项特殊选项说明

| 事项 | 特殊选项 | 说明 |
|------|----------|------|
| 嫁娶 | 新娘/新郎生肖 | 避开生肖相冲日 |
| 安葬 | 山向、仙命 | 选择坐山方向，避开仙命相冲 |
| 修造 | 坐山方向 | 根据坐山选择吉日 |
| 作灶 | 灶向、灶位 | 选择灶口方向和灶位 |
| 安床 | 床位朝向 | 选择床位方向 |
| 出行 | 出行方向 | 选择出行方向 |
| 开业 | 店铺坐向 | 根据店铺坐向选择 |
| 入宅 | 新居坐向 | 根据新居坐向选择 |

#### 选项填写建议

- **生肖选择**：根据农历年份选择，注意立春为年界
- **山向选择**：使用罗盘测量，精确到24山向
- **方向选择**：以房屋中心为基准，面向方向为向

### 4.3 事主信息输入与八字分析

#### 添加事主

1. 点击"添加事主"按钮
2. 填写信息：
   - **姓名**：可选，用于标识
   - **性别**：必须，影响大运计算
   - **出生日期**：公历日期

## 5. 核心理论依据

### 5.1 正五行择日法

#### 5.1.1 补龙

**概念**：日课五行补益山家（坐山）五行。

**原理**
- 山家五行不足者，用日课补之
- 山家五行过旺者，用日课泄之
- 以达五行平衡为要

**应用**
- 安葬：补龙脉之气
- 修造：补坐山之运
- 作灶：补灶位之方

#### 5.1.2 扶山

**概念**：月令、日支扶助山家。

**原理**
- 月令五行生扶山家为吉
- 日支五行与山家比和为吉
- 避免月令、日支冲克山家

**应用**
- 春季（木旺）：宜选木山、火山
- 夏季（火旺）：宜选火山、土山
- 秋季（金旺）：宜选金山、水山
- 冬季（水旺）：宜选水山、木山

#### 5.1.3 相主

**概念**：日课与事主八字相合。

**原理**
- 日课五行补益事主喜用神
- 避免日课冲克事主八字
- 考虑事主生肖、命宫

**应用**
- 嫁娶：男女八字相合
- 安葬：仙命与日课相合
- 修造：宅主八字与日课相合

#### 5.1.4 避凶

**概念**：避开各种凶煞。

**凶煞类型**
1. **年煞**：太岁、岁破、三煞等
2. **月煞**：月建、月破、月刑等
3. **日煞**：杨公忌日、红砂日、受死日等
4. **时煞**：日破时、五不遇时等

**避凶原则**
- 大事避大煞，小事避小煞
- 吉神能制凶煞者可化
- 凶煞重叠者坚决不用

## 6. 常见问题（FAQ）

### 6.1 为什么同一天在不同软件中四柱不同？

**原因分析**

1. **节气计算精度不同**
   - 不同软件使用的节气算法可能有差异
   - 立春、立夏等交界时刻前后容易有差异

2. **年界标准不同**
   - 有些软件以春节为年界
   - 本软件以立春为年界（符合传统择日学）

3. **历法数据来源不同**
   - 不同软件使用的农历数据表可能不同
   - 古代历法修正数据可能有差异

**本软件的处理方式**

- 使用 **sxtwl**（天文算法）和 **lunar-python**（数据表）双库校验
- 以 **立春** 为年界，以 **节气** 为月界
- 发现差异时会发出警告，用户可自行判断

**建议**

- 重要事项建议以通书或专业择日师为准
- 本软件结果仅供参考

### 6.2 事主信息必须填写吗？

**答案**：不是必须的，但建议填写。

**不填写的影响**
- 软件仍可进行择日
- 但不会进行八字匹配分析
- 无法判断日课是否与事主相合

**填写的好处**
- 提供更个性化的择日建议
- 避开与事主八字相冲的日期
- 选择与事主喜用神相符的日课

**适用场景**

| 场景 | 建议 |
|------|------|
| 为他人择日 | 建议填写事主信息 |
| 自己使用 | 建议填写自己的八字 |
| 快速查询 | 可不填写，先大致筛选 |
| 学习研究 | 可不填写 |

### 6.3 评分低的日期能否使用？

**一般原则**

| 等级 | 建议 |
|------|------|
| ❌凶（0-20分） | 坚决不用 |
| ⭐平（21-40分） | 非急勿用 |
| ⭐⭐次吉（41-60分） | 可用，但非最佳 |
| ⭐⭐⭐吉（61-80分） | 推荐使用 |
| ⭐⭐⭐⭐大吉（81-100分） | 优先使用 |
| ⭐⭐⭐⭐⭐上吉（100+分） | 最佳日期 |

**特殊情况**

1. **时间紧迫**
   - 如果必须在近期办理，可选择次吉日
   - 同时选择吉时，以时补日

2. **吉神化解**
   - 有些凶煞可用吉神化解
   - 如天德、月德可化解大部分凶煞

3. **专业判断**
   - 评分只是参考
   - 专业择日师可能根据具体情况灵活处理

### 6.4 "次吉"等级的含义

**定义**

"次吉"（⭐⭐，41-60分）表示该日期：
- 没有严重凶煞
- 五行基本平衡
- 但不是最佳选择

**适用情况**

- 时间选择范围有限
- 其他日期有更严重问题
- 配合吉时使用

**注意事项**

- 次吉日可用，但效果不如"吉"或"大吉"
- 重要事项建议优先选择更高等级的日期
- 使用次吉日时可加强其他方面的准备

### 6.5 软件能否用于阴宅（安葬）？

**答案**：可以。

**安葬择日特点**

1. **特殊要求**
   - 需要考虑仙命（逝者八字）
   - 需要考虑坐山方向
   - 需要避开重丧、三丧等凶煞

2. **软件支持**
   - 提供"安葬"事项类型
   - 支持输入仙命信息
   - 支持选择山向
   - 包含安葬专用神煞检查

**注意事项**

- 安葬是大事，建议咨询专业风水师
- 软件结果仅供参考
- 各地风俗不同，需结合当地习惯

### 6.6 如何验证软件准确性？

**验证方法**

1. **与通书对比**
   - 选择几个已知日期
   - 对比软件结果与通书（如《协纪辨方书》）
   - 注意：不同通书可能有差异

2. **与在线工具对比**

   - 对比四柱、神煞等信息
   - 注意：在线工具也可能有误

3. **实际验证**
   - 记录使用软件选择的日期
   - 观察后续发展
   - 积累经验

4. **专业咨询**
   - 重要事项咨询专业择日师

   - 学习专业判断方法
