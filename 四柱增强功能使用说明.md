# 四柱增强功能使用说明

## 一、功能概述

四柱计算器现在集成了八字工具整合模块的深层分析功能，可以为sizhu字典添加以下分析结果：

1. **十二长生状态**：计算日主在各柱的十二长生状态及力量系数
2. **纳音五行**：计算四柱的纳音五行
3. **地支关系**：检查六合、六冲、六害、三刑、三合、三会等地支关系
4. **五行分数**：使用加权算法计算各五行的精确分数

## 二、新增函数

### 2.1 `analyze_sizhu_advanced(sizhu)`

深度分析四柱八字，返回分析结果字典。

**参数**：
- `sizhu`: `calculate_sizhu()`返回的字典

**返回值**：
```python

    '十二长生': {
        'year': {'状态': '长生', '力量系数': 1.0},
        'month': {'状态': '沐浴', '力量系数': 0.8},
        'day': {'状态': '冠带', '力量系数': 0.9},
        'hour': {'状态': '临官', '力量系数': 1.0}
    },
    '纳音': {
        'year': '海中金',
        'month': '炉中火',
        'day': '大林木',
        'hour': '路旁土'
    },
    '地支关系': {
        '六合': [{'地支': '子丑', '合化': '土'}],
        '六冲': ['子午'],
        '六害': ['子未'],
        '三刑': [{'地支': '寅巳', '类型': '无恩之刑'}],
        '三合': '水',  # 或 False
        '三会': '木'   # 或 False
    },
    '五行分数': {
        '金': 2.25,
        '木': 2.15,
        '水': 0.20,
        '火': 2.25,
        '土': 2.35
    }
}
```

### 2.2 `enhance_sizhu(sizhu)`

增强sizhu字典，将深度分析结果直接附加到sizhu字典中。

**参数**：
- `sizhu`: `calculate_sizhu()`返回的字典

**返回值**：
增强后的sizhu字典，包含新增的`'深度分析'`键。

## 三、使用示例

### 3.1 基本使用

```python
from modules.四柱计算器 import calculate_sizhu, enhance_sizhu
from datetime import date

# 计算四柱
sizhu = calculate_sizhu(date(2024, 5, 20), 12, 0)

# 增强分析
enhanced = enhance_sizhu(sizhu)

# 访问深度分析结果
if '深度分析' in enhanced:
    analysis = enhanced['深度分析']
    
    # 查看十二长生
    print("十二长生:", analysis['十二长生'])
    
    # 查看纳音
    print("纳音:", analysis['纳音'])
    
    # 查看地支关系
    print("地支关系:", analysis['地支关系'])
    
    # 查看五行分数
    print("五行分数:", analysis['五行分数'])
```

### 3.2 结合喜用神计算

```python
from modules.四柱计算器 import calculate_sizhu, enhance_sizhu
from modules.喜用神计算器 import calculate_xishen_yongshen
from datetime import date

# 计算四柱并增强

# 获取五行分数
wuxing_score = enhanced['深度分析']['五行分数']

# 计算喜用神（使用加权分数）
xishen, yongshen = calculate_xishen_yongshen(enhanced)

print(f"喜用神: {xishen}, 用神: {yongshen}")
print(f"五行分数: {wuxing_score}")
```

## 五、应用场景

### 5.1 日主旺衰判断

```python
from modules.四柱计算器 import enhance_sizhu

enhanced = enhance_sizhu(sizhu)
wuxing_score = enhanced['深度分析']['五行分数']
day_wuxing = get_gan_wuxing(sizhu['day_gan'])
day_score = wuxing_score[day_wuxing]

if day_score > 6.0:
    print("日主太旺")
elif day_score > 3.0:
    print("日主旺")
elif day_score < 2.0:
    print("日主弱")
else:
    print("日主中和")
```

### 5.2 地支关系分析

```python
from modules.四柱计算器 import enhance_sizhu

enhanced = enhance_sizhu(sizhu)
relations = enhanced['深度分析']['地支关系']

# 检查是否有三合局
if relations['三合']:
    print(f"三合{relations['三合']}局")

# 检查是否有六冲
if relations['六冲']:
    print(f"六冲: {relations['六冲']}")

# 检查是否有三刑
if relations['三刑']:
    for xing in relations['三刑']:
        print(f"三刑: {xing['地支']} ({xing['类型']})")
```

### 5.3 十二长生分析

```python
from modules.四柱计算器 import enhance_sizhu

enhanced = enhance_sizhu(sizhu)
zhangsheng = enhanced['深度分析']['十二长生']

# 查看日主在各柱的状态
for pillar, info in zhangsheng.items():
    print(f"{pillar}: {info['状态']} (力量{info['力量系数']})")

# 判断日主是否得令
month_state = zhangsheng['month']['状态']
if month_state in ['长生', '沐浴', '冠带', '临官', '帝旺']:
    print("日主得令")
else:
    print("日主失令")
```

## 六、注意事项

1. **向后兼容**：`enhance_sizhu`函数不会修改原始sizhu字典，而是返回一个新的增强字典。

2. **错误处理**：如果八字工具整合模块不可用，`analyze_sizhu_advanced`会返回空字典，`enhance_sizhu`不会添加'深度分析'键。

3. **性能考虑**：深度分析会增加计算时间，建议在需要时再进行增强，而不是每次都调用。

4. **纳音计算**：纳音计算需要完整的干支对（如"甲子"），如果sizhu中只有天干地支分开的键，可能无法正确计算纳音。
